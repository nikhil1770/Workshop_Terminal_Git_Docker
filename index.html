Phase 2: Containerization

Navigation Commands
pwdPrint current directory
ls -laList all files (including hidden)
cd ..Move up one folder
mkdir [name]Create new folder
The Golden Workflow
git initStart a local repo
git add .Stage all changes
git commit -m "msg"Save snapshot
git push origin mainUpload to GitHub
Container Management
docker psShow running containers
docker imagesList downloaded blueprints
docker run -it [img]Start interactive container
Internal Architecture

Git stores data in three areas:

  • Working Tree: Files you see.
  • Staging Area: The "Index" of next commit.
  • Repository: Permanent snapshots in .git
